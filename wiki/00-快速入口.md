## 决策树

```
你要做什么？
│
├─ 了解系统
│   └─ 走 01 篇架构总览
│
├─ 替换资源层（接入新的设计系统）
│   └─ 走 03 篇资源层替换指南
│
├─ 生成页面
│   └─ 走 04 篇页面生成流程
│
└─ 维护/新增内容
    ├─ 新增 Layout → 走 05 篇新增 Layout 指南
    ├─ 新增 Block → 走 05 篇新增 Block 指南
    └─ 新增 Component → 走 05 篇新增 Component 指南
```

## 常见任务一句话指南

| 场景 | 怎么做 |
|------|--------|
| 我要接入一个新的设计系统 | 03 篇 → 告诉 AI "帮我把 [设计系统名] 接入为新资源集" |
| 我要切换当前使用的设计系统 | 03 篇 → 修改 config.json 的 active 字段 |
| AI 生成的页面不对 | 04 篇 → 告诉 AI 哪里不对 |
| AI 生成的规范我审什么 | 看变体和状态是否和设计稿一致 |
| 我想看看系统里有哪些组件 | 02 篇 → 查设计系统资源概览 |
| 我不确定该走哪个流程 | 本文档 → 决策树 |
