# 02 | 资源层配置指南

> 这一篇说明如何配置和管理设计系统资源集。

---

## 什么时候走这个流程？

当你需要：
- 添加新的设计系统资源集
- 切换当前使用的设计系统
- 修改资源集的配置

---

## config.json 结构

```json
{
  "active": "harmony",
  "resources": {
    "harmony": {
      "path": ".resources/harmony",
      "projectRoot": "harmony-ui-playground"
    }
  }
}
```

---

## 每一步谁做什么

🤖 = AI 独立完成
👤 = 设计师独立完成
🤝 = AI 和设计师协作

### 添加新资源集

🤝 设计师提供设计系统信息，AI 协助配置

### 切换资源集

🤖 AI 修改 config.json 的 active 字段

### 配置验证

🤖 AI 验证配置是否正确

---

## 核心工具

### add-resource — 添加资源集

**做什么：** 在 config.json 中注册新的设计系统资源集

**输出物：** 更新后的 config.json
