# harmony-ui-playground

Vite + React + shadcn/ui playground skeleton for Harmony style components.

## 当前状态

- 项目骨架已建立
- `src/component` 已完成按组件拆分与统一目录
- 可直接在该目录下维护和新增组件

## 快速开始

```bash
npm install
npm run dev
```

## 目录结构（核心）

- `.storybook/`: Storybook 配置
- `src/component/`: 组件目录（统一）
- `src/styles/devui-tokens.css`: 设计 token
- `components.json`: shadcn 配置
